# E-Commerce Sales and Customer Analytics (SQL Project)

## Project Overview
This project focuses on analyzing an e-commerce platform’s sales, customers, and order behavior using SQL.  
The objective is to extract meaningful business insights related to revenue, customer activity, product performance, and ordering trends.

This project demonstrates practical SQL skills used in real-world data analysis scenarios.

---

## Dataset Description
The dataset represents a typical e-commerce system with multiple relational tables.

Tables included:
- users – customer details
- orders – order information and timestamps
- products – product catalog
- order_items – mapping between orders and products
- payments – payment records

Each table is designed with primary keys and foreign key relationships to maintain data integrity.

---

## Key Analysis Questions
- What is the total revenue generated?
- What are the daily and monthly sales trends?
- Which products generate the highest revenue?
- Who are the most valuable customers?
- How many repeat customers are there?
- What is the average order value?
- Which payment methods are most used?
- Which orders were cancelled?
- How frequently do customers place orders?
- Which product categories perform best?

This project includes more than 30 intermediate to advanced SQL queries.
